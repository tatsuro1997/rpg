## Overview

人生の豊かさは過去の経験によって大きく変わるという『[DIE WITH ZERO](https://amzn.asia/d/0TnGUgI)』からの着想。
人生の経験値を可視化し、成長や豊かな人生促進に寄与することが開発者の意図である。

## Getting Started

First, create node_modules:

```bash
npm i
```

Second, run the development server:

```bash
npm run dev
```
